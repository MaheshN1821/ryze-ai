# Ryze.Ai

## This project is a redesigned version of the Get-Ryze.ai website. The goal was to improve visual appeal, usability, and clarity while keeping the design modern, clean, and professional. The redesign focuses on clearly communicating Ryze’s value proposition and improving user engagement.

## Website live link: https://ryze-ai-seven.vercel.app/

# Objective

## Redesign the Get-Ryze.ai website to enhance branding, layout, and user experience.
## Create a clear and compelling homepage along with at least two inner pages.
## Ensure the website is responsive, accessible, and easy to navigate.

# Pages Covered

## Homepage – Introduces Ryze, its value proposition, and key features with strong CTAs.
## Features Page – Explains core functionalities and benefits in detail.
## Pricing Page – Clearly presents plans, pricing structure, and comparisons.

# Design Approach

## Followed a modern, minimal, and professional visual style.
## Used consistent typography and spacing to improve readability.
## Applied a clean color palette aligned with a SaaS/AI brand identity.
## Focused on clarity and hierarchy to guide users through content naturally.

# User Experience (UX) Decisions

## Designed clear and prominent call-to-action buttons such as “Book a Demo” and “Get Started”.
## Simplified navigation for faster access to important sections.
## Ensured mobile-first responsiveness across all screen sizes.
## Maintained accessibility best practices including contrast, spacing, and readable font sizes.

# Modular Structure

## Header / Navigation – Reusable navigation with clear links and CTA.
## Hero Section – Strong headline, subheading, and primary action.
## Features Section – Modular feature cards that can be easily updated.
## Testimonials Section – Social proof to build trust and credibility.
## Pricing Section – Flexible layout for adding or modifying plans.
## Footer – Contains quick links, contact information, and branding.

# Tech Stack

## Frontend framework/library used for building the UI.
- React.js
- Tailwind CSS
- Javascript

## Deployed using Vercel hosting platform.

# How to Run Locally

## Clone the repository from GitHub.
## Install dependencies using the package manager.
## Run the development server to view the project locally.
